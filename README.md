# Movie-Search-App

## 개요

-   영화 검색 사이트에 다양한 기능을 추가 함으로써 React, Redux, Typescript를 학습 하기 위함
-   회원가입 / 로그인 / 댓글 / 추천 / 검색 등등

## 기술스팩 (필요한 스팩은 진행하면서 추가)

-   Redux || Redux Toolkit || recoil ? (상태관리는 추후 선택..)
-   TypeScript
-   Tailwind CSS
-   webpack V5

### Back End

-   Express
